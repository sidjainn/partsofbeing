# partsofbeing

Static single-page artwork built with plain HTML, CSS, and D3.

## Run locally

1. Open a terminal in the project folder

2. Start a local server: python3 -m http.server 8000

3. Visit [http://localhost:8000](http://localhost:8000).

You can also open `index.html` directly in a browser, but serving the folder locally is more reliable for asset loading.
